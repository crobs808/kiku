# Kiku v0.1.4 - Initial Alpha (Pre-Release)

Release date: 2026-04-20

## Summary
Kiku is now available as an initial macOS desktop release focused on local-first live transcription and translation workflows for Japanese/English testing.

## Highlights
- Local-first live captions in a desktop app (Tauri + Rust + React).
- Input source controls for microphone, system audio, or both.
- Built-in model manager for Whisper-family downloads, switching, and deletion.
- Session controls for start/stop plus save/discard transcript flow.
- Live transcript stream with source labels and follow-live behavior.
- Streaming translation mode toggle (faster partial output with correction behavior).
- Cloud provider settings UI for optional Google Cloud ASR/translation integration.
- macOS system-audio permission guidance flow with restart handling.
- Japanese Phrase Lab flyout for transcription validation using clickable phrase audio and JP/EN reference text.

## Known Limitations
- This is an early pre-release and still prototype-oriented.
- Model downloads and Phrase Lab remote phrase audio depend on network connectivity.
- Desktop packaging and verification are macOS-first at this stage.
